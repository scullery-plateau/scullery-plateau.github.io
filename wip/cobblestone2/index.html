CobbleStone: Pixel Art Map Tiles

Cobblestone is a tool meant to build printable and downloadable dungeon tiles using pixel art.

Each 1 inch space on the page is a 16x16 pixel tile. These tiles are defined by using different ascii characters of your choice to represent each color in the tile. The maps and tile sheets are defined the same way - using different ascii characters to represent each tile. A mapping is also defined to map each of the chosen characters to a tile, a palette, and the orientation of that tile in the map.

To see the tool in action, download the map file below, then load that file back into the app. Make changes as you like, or create a file of your own.

  • When making changes to the file, tile sheets and tiles can use any character other than double quotes (") and colons (:).
  • Orientations are any combination of "turnleft", "turnright", "flipover", and "flipdown". (to turn a tile 180 degrees, use flipover and flipdown together).
  • Tile sheets need to be no more than 8 tiles wide and 10 tiles long, as they are made printable on 8 1/2" x 11" sheet size

Press 'Print' once file is loaded to print or save as pdf to print later. Click on individual images to download as png files.
