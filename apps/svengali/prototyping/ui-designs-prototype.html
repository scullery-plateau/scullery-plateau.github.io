Image Tab

Image gallery management and usage tracking.

Thumbnail Filename
hero.png
mage.png
dragon.jpg
Usage Notes:
  • No per-row delete buttons. Use "Remove Unused" to bulk-remove unreferenced images.
  • An image is "in use" if its filename appears in any value-mode Image.data field OR in any ImageData-type column in state.data rows.
  • Duplicate filenames on load are rejected with an inline message.
