## Coding style / flavor directives

* All javascript files should be namespaces using the skill `https://gizmo-atheneum.github.io/structure/importnamespace/SKILL.md`
* Any stateless React components should be implemented as functions
* Any stateful React components should be classes that extend `React.Component`
* css / style uses bootstrap and css files in `../../../style`
  * see `../index.html` for specific version of bootstrap
* use `dialog.js` for modals
