## Menu Bar Structure

```
┌──────────────────────────────────────────────────────────────────────────────────────────┐
│ [⬇] All │ [📄] Template │ [📊] Data │ [🖨] Print │ [📂] Load Template │ [📂] Load Data │
└──────────────────────────────────────────────────────────────────────────────────────────┘

Icon Legend:
  ⬇  = fa-download      (Download All)
  📄 = fa-file         (Download Template)
  📊 = fa-table        (Download Data)
  🖨  = fa-print       (Print)
  📂 = fa-folder       (Load Template / Load Data)
```
